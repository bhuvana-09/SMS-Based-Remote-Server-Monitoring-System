📡 SMS-Based Remote Server Monitoring System

📌 Project Description

The SMS-Based Remote Server Monitoring System is a web application that monitors server performance such as CPU usage, memory usage, and disk usage.

When any parameter exceeds its threshold value, the system automatically generates an alert and sends an SMS notification to the administrator.

🛠️ Technologies Used

PHP

MySQL

HTML & CSS

Cron Jobs

SMS API Integration

🗄️ Database Tables

administrators – Stores admin details

servers – Stores server information

metrics – Stores server performance data

alerts – Stores alert records

thresholds – Stores threshold values

settings – Stores system configuration

⚙️ How It Works

Server metrics are collected periodically.

Values are compared with threshold limits.

If limits are exceeded, an alert is generated.

SMS notification is sent to the administrator.

🚀 Features

Real-time server monitoring

Automatic SMS alerts

SQL queries (joins, subqueries, aggregate functions)

Database normalized up to 3NF

📖 Conclusion

This project demonstrates practical implementation of DBMS concepts, PHP-MySQL integration, and automated alert systems for proactive server management.
